# 🏗️ Construction Field Management App

A responsive React.js web application built for field managers to track project progress and file Daily Progress Reports (DPR).

## 🚀 Features
- **Secure Mock Authentication**: Access control using specialized Context providers.
- **Project Dashboard**: Card-based overview of construction projects with status-coded badges.
- **Dynamic DPR Form**: 
    - Context-aware project selection.
    - Real-time image preview (up to 3 photos).
    - Integrated date picker and weather tracking.
    - Comprehensive input validation with error messaging.
- **Responsive UI**: Mobile-first approach ensuring usability on construction sites (375px+) up to desktop (1280px+).

## 🛠️ Tech Stack
- **Framework**: React 18 (Vite)
- **Routing**: React Router v6
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: React Context API & Hooks
